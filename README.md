# Bank Account Management System

A desktop banking application developed using **Java**, **JavaFX**, and **Object-Oriented Programming (OOP)** principles.

## Overview

This project simulates a real-world banking system with a graphical user interface that allows users to manage bank accounts efficiently.

## Features

* Secure Login System
* Create New Accounts
* Deposit Money
* Withdraw Money
* Transfer Funds Between Accounts
* View Account Details
* Generate Reports
* User-Friendly JavaFX Interface

## Technologies Used

* Java
* JavaFX
* OOP Concepts
* Scene Builder
* Git & GitHub

## Project Structure

* `model/` → Classes representing bank accounts and transactions
* `controller/` → Handles GUI events and user actions
* `service/` → Business logic and operations
* `resources/` → FXML views and UI files

## Purpose

This project was built as an educational software engineering project to apply programming fundamentals, GUI development, and clean code organization.
