# Overview

Nexo is a modern desktop banking management system developed using Java and JavaFX.

The project simulates real-world banking workflows including:
- Account Management
- Financial Transactions
- Loan Systems
- Reporting & Analytics
- Monthly Financial Updates
- Dashboard Visualization

The application is designed using clean architecture principles and Object-Oriented Programming concepts.

# Architecture

The project follows a layered architecture structure:

```text
model/
repository/
service/
controller/
resources/
```

| Layer | Responsibility |
|---|---|
| model | Business entities and account models |
| repository | File handling and data persistence |
| service | Banking business logic |
| controller | JavaFX interaction logic |
| resources | FXML views and assets |
